# Siklus: Paket Malam 📦🛵

Sebuah game 3D Indie Horror Survival tentang seorang kuris COD yang terpaksa mengambil shift malam demi melunasi utang, tanpa menyadari bahwa pelanggannya bukanlah manusia.

## 🎮 Informasi Game
* **Engine:** Unity 2022
* **Genre:** 3D Indie Horror, Survival
* **Gaya Visual:** Retro PS1 (Low-Poly)

## 🔄 Core Loop
1. **Bangun & Cek Ordran:** Pemain mengecek target paket malam itu.
2. **Naik Motor & Ekspedisi:** Membawa semua paket sekaligus.
3. **Perjalanan Berantai:** Mengantar paket ke area pinggiran kota yang gelap secara berurutan.
4. **Pulang & Tidur:** Kembali ke rumah untuk mengulang siklus berikutnya.
